The external documentation of the project can be found in "./Documentation/",
there is a markdown source file where the documentation was written,
and the output PDF is called "DocumentacionProyecto.pdf".

The diagram of the project is located in "./Diagram/class-diagram.png",
and also declared as a figure in the external documentation PDF.

The source code of the project can be found in the "Code/" folder, 
there are the header files, the main file, and all the rest of the modules.

You need to have installed make, mold linker and clang compiler to 
build these source files into a binary file.

Important note: This project was developed on Linux, probably will have errors
compiling or running on windows.
